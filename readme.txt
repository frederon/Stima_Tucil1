Tugas Kecil 1 Strategi Algoritma (IF-2211)

Tugas mata kuliah Strategi Algoritma untuk membuat program yang dapat mencari solusi cryptarithmetic dengan algoritma brute force

Persyaratan 
-----------------------------
Pastikan Anda sudah mempunyai Python minimal versi 2.7

Cara Menggunakan
-----------------------------
1) Pergi ke folder src dengan cara mengetikkan pada terminal:

cd ./src

2) Jalankan program dengan cara mengetikkan pada terminal:

python main.py

3) Setelah program berjalan, masukkan nama file yang ingin dicari solusinya.
Contohnya jika solusinya berada pada folder test sebelumnya maka:

Masukkan nama file: ../test1.txt

atau, jika file txt sudah berada pada folder src maka

Masukkan nama file: test1.txt

4) Untuk menambahkan persamaan baru dapat dilakukan pada file teks yang sama/sebelumnya atau membuat file baru. Pastikan format penulisan persamaan benar

Contoh input/output
-----------------------------
Masukkan nama file: test1.txt

NUMBER 
NUMBER+
------ 
PUZZLE 

201689
201689+
------
403378

Total waktu: 6.988 detik
Total tes: 1489456
